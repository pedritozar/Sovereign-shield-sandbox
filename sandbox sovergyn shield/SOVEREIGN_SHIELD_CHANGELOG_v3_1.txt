=======================================================================
 OBSTÁCULOS TÉCNICOS — REGLAS PARA IAs
=======================================================================

1. PapaParse → parseFloat() siempre antes de operar.
2. Leaflet → validar typeof L !== 'undefined'.
3. Animaciones LOCAL (CSS/SVG), Firebase recibe solo resultado.
4. API Key NUNCA en código. localStorage('ss_anthropic_key').
5. Tesorería → SIEMPRE transaction(), nunca set() directo.
6. setStatus() NO EXISTE en index.html → usar agregarEvento(msg,tipo).
7. cerrarTrimestre() NO es async → llamar funciones async sin await.
8. No usar clases CSS inexistentes (.panel-title,.txt-muted,.btn-green).
   Usar <h3>, estilos inline, border-color con variables CSS.
9. Leaflet NO tiene markerPatagonica.disable()/.enable() →
   usar setStyle({opacity, fillOpacity}).
10. Chart.js → setTimeout 1200ms esperando Firebase.
11. IndexedDB → initAcademyDB() retorna Promise, siempre await.
12. Usar var (no const/let) para consistencia con el resto del código.
